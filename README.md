## Psi: Tweaks And Additions

PsiTweaks is an addon mod that expands the magical programming mod Psi, providing a deeper experience and extending its progression well into the late game.

It adds numerous spell pieces, new programming types, higher-tier CADs and spell bullets, casting-support equipment, a research system, and various quality-of-life improvements. When Mekanism is installed, players can also access industrial production methods for Psi materials, dedicated machines, and MekaSuit modules.

Documentation for these features is included in the in-game Encyclopaedia Psionica through Patchouli.

### Main Features

#### Expanded Spell Programming

*   Numerous spell pieces for combat, flight, string processing, matrix operations, flow control, and more
*   New types such as Item, Block, String, and Matrix
*   Storage of numbers, strings, matrices, and other values in CAD memory
*   A research system for unlocking certain advanced spell pieces
*   Quality-of-life improvements for the Spell Programmer

In addition to straightforward offensive spells, these features allow you to create complex spells that work with item and block data, as well as programs designed for automation.

#### CADs, Spell Bullets, and Equipment

*   Several new families of CAD materials and high-performance CAD assemblies
*   Higher-tier versions of the various spell bullet types
*   Useful magic items such as Inline Casters and automatic casting devices
*   Spell Magazines that allow spell bullets to be stored and swapped
*   New and powerful equipment designed for Magicians

Higher-tier equipment makes it practical to use large-scale, high-cost spells that would be difficult to handle with standard Psi equipment.

#### Mekanism Integration

PsiTweaks strongly recommends installing Mekanism and adds various features that connect Psi progression with industrial machinery.

*   Production of Psi materials using the Metallurgic Infuser
*   The Program Researcher, Material Mutator, and Sculk Eroder
*   The Psionic Generator, which generates energy by consuming a linked player's Psi
*   MekaSuit modules that increase Psi regeneration, maximum Psi capacity, and spell damage

When Mekanism is not installed, Mekanism-specific machines and modules are not added. Alternative recipes are provided for certain materials and research programs.

### Required and Recommended Mods

**Required**

*   Psi
*   Curios API
*   Patchouli

**Strongly Recommended**

*   Mekanism
*   Mekanism Generators
*   JEI

**Optional Integrations**

*   Create Aeronautics / Sable (1.21.1)
*   Tinkers' Construct (1.20.1)

Please note that some features may differ depending on the Minecraft version and the other mods installed.
