## Psi: Tweaks And Additions

Psiに追加要素やMekanismとの連携レシピを追加するmodです。
前提modとしてMekanism, Mekanism Generators, Psi が必要です。

- 主な追加要素

被弾時のサイオン量減少無効化

サイオン回復のクールダウン無効化

サイメタル装備の上位互換追加

Mekanismの冶金注入によるサイメタルなどの作成レシピ追加

Psi1.20.1よりフラッシュリングのバックポート

#### v0.5.1

サーバー側でクラッシュする問題を修正

#### v0.6.0

- 改良術式弾を追加 
    * 消費コストが元の85%になる
  

- 作動式:FE供給 を追加


- 作動式:時間加速 を追加
    * Just Dire ThingsやTime in a bottleのようなtick加速を可能(ただし現在はCADの性能の都合上8倍速まで)


- サードアイデバイス を追加、CADの32マス制限を解除
    * 実装に伴い、Raycastの射程を256ブロックに延長
    * 取得子: 近くのエンティティ などの範囲は変更されていない

- カオティックサイメタルCAD, サイオニック合金CADのモデルが正しく実装されるように

#### v0.6.1

- Psi実績の日本語化


- ムーバルスーツヘルメットにセンサーを取り付けられるように

#### v0.7.0

- CADアセンブリ三種類を追加
    * フラッシュメタルのCAD素体
    * ヘビーサイメタルのCAD素体 α
    * ヘビーサイメタルのCAD素体 β


- 中間素材大量追加


- Pψ-E 燃料追加
    * Mekanism Generatorsのガス発電機で使用可能


- 作動式: フォノンメーザー追加
    * 高威力の光線を前方に放つ攻撃用魔法
  

- 作動式: 超位注入追加
    * 残響の欠片をサイオニックエコーに変換する。フラッシュメタル以上のCADが必要
    * サイオニックエコーはヘビーサイメタルやPψ-E 燃料の作成に必要


- CAD分解台を追加
    * スニーク右クリックでCADを分解し、コンポーネントを回収可能

- 共鳴術式弾、昇華術式弾を追加
    * それぞれもとの術式弾の70%と50%のコストで使用可能

#### v0.7.1 (plan)

- 作動式: 分子ディバイダー 追加
    * 三点で領域を作り内部の者にダメージを与える

- 作動式: ブロック破壊(幸運)、ブロック破壊(シルクタッチ) 追加

- CAD素体の規模を大幅に強化
    * 合わせて一部術式の必要規模も上昇
  
- Psi本体のブロックの適正ツール、サイメタルブロックのルートテーブル修正
